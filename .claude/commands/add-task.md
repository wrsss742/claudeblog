以下の引数からタスク名と期限を受け取り、`tasks.md` にタスクを追記してください。

引数: $ARGUMENTS

## 手順

1. 引数を解析して「タスク名」と「期限（YYYY-MM-DD形式）」を取得する
2. 期限が `YYYY-MM-DD` 形式でない場合はエラーメッセージを表示して処理を中止する
3. 今日の日付を `YYYY-MM-DD` 形式で取得する
4. 以下のフォーマットで1行を作成する：
   ```
   - [ ] タスク名 (期限: YYYY-MM-DD) [登録: 今日の日付]
   ```
5. `tasks.md` を編集して追記する：
   - `tasks.md` が存在しない場合は以下のヘッダーを付けて新規作成する：
     ```markdown
     # タスク管理

     ## タスク一覧
     ```
   - `## タスク一覧` セクションが存在する場合はその末尾に追記する
   - セクションが存在しない場合はファイル末尾に `## タスク一覧` ごと追記する
6. 追記した内容を確認してユーザーに報告する
